Search Syntax

Powerful Lucene query syntax for searching bookmarks

Quick Reference

title:python

Title contains "python"

tags:tutorial

Has tag "tutorial"

title:python AND tags:web

Both conditions must match

tags:react OR tags:vue

Either tag matches

title:"machine learning"

Exact phrase in title

tags:java*

Tags starting with "java"

Searchable Fields

title

Search in bookmark titles

title:python title:"web development"

description

Search in bookmark descriptions

description:tutorial

tags

Search bookmark tags

tags:python tags:web AND tags:javascript

url

Search in URLs

url:github.com

Operators

AND

Both terms must be present

title:python AND tags:tutorial

OR

Either term can be present

tags:react OR tags:vue

NOT

Exclude results with this term

tags:python NOT tags:django

Grouping ( )

Group conditions together

(tags:python OR tags:java) AND tags:web

Special Features

Phrase Search

Use quotes for exact phrases

title:"machine learning"

Wildcards

Use * for zero or more characters

tags:java*

Matches: java, javascript, javaee, etc.

Real-World Examples

Find all Python tutorials

title:python AND tags:tutorial

Find React or Vue resources

tags:react OR tags:vue

Find web development resources, excluding WordPress

tags:web NOT tags:wordpress

Find GitHub repositories about machine learning

url:github.com AND title:"machine learning"

Find all JavaScript-related tags

tags:java*

Complex query with grouping

(tags:python OR tags:java) AND (tags:web OR tags:api) NOT tags:deprecated

💡 Pro Tips

  • Searches are case-insensitive
  • Use wildcards (*) for flexible matching
  • Combine multiple fields for precise results
  • Use parentheses for complex queries
  • Click tags on bookmarks to auto-search for that tag
